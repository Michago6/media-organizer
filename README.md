# Media Organizer

A collection of shell scripts for organizing and maintaining my photo and video libraries.

## About

This repo contains shell scripts I built for myself to automate the tedious parts of organizing my photos and videos. These scripts help me organize media into dated folder structures (YYYY/MM), detect duplicates, and fix files with mismatched EXIF metadata. They're tailored to my specific workflow and not really designed as general-purpose tools.



## How to Use

Each script includes built-in usage instructions and safety warnings. Just run them and follow the prompts:

```bash
./organize.sh
./fix-wrongly-dated.sh
```

Check the ATTENTION and TLDR sections at the top of each script before running.

## Notes

- These are personal tools built for my specific photo organization needs
- They handle common image and video formats
- Always ensure backups exist before using destructive scripts like `organize.sh`
